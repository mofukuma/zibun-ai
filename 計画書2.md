# Claude AIチャットプログラム(Streamlit版) 開発計画書

## 1. 要件(このプログラムで実現したいこと)

### やりたいこと
- ブラウザ上でClaude AIと日本語で会話できるアプリを作る
- メッセージを入力して送ると、AIが返事をくれる
- 過去のやり取りが画面に残り、AIも会話の流れを覚えている
- 画面がきれいで使いやすい

### 作り方のルール
- プログラムは1つのファイル(chat_2.py)にまとめる
- ファイルの先頭、関数の前、難しい処理の前に日本語でコメントを書く
- Streamlitという画面作成ツールを使う
- APIキー(パスワードのようなもの)は.envファイルで安全に管理する

## 2. 使う技術

### プログラミング言語
- **Python** - AIとのやり取りや画面表示を全て担当

### 使うツール・部品

| ツール名 | 何に使うか |
|---------|-----------|
| **Streamlit** | ブラウザに表示するチャット画面を作る道具。Pythonだけで画面が作れる |
| **anthropic** | Claude AIと通信するための公式の部品 |
| **python-dotenv** | .envファイルからAPIキーを読み込む部品 |

### 前回(chat_1.py)との違い

| 項目 | chat_1.py(前回) | chat_2.py(今回) |
|------|----------------|----------------|
| 画面作成 | Flask + HTML/CSS/JavaScript | Streamlitだけ |
| コードの量 | 多い(HTMLを書く必要あり) | 少ない(Streamlitが画面を自動生成) |
| チャット画面 | 自分で作った | Streamlit標準のチャット部品を使う |
| APIキー管理 | 環境変数を直接使用 | .envファイルで管理 |
| 会話の記憶 | ブラウザ上の表示のみ | AIにも過去の会話を送って記憶させる |

## 3. 機能

### 3.1 チャット画面の表示
- ブラウザにチャット画面が表示される
- 画面の要素:
  - ページタイトル(「Claude AIチャット」)
  - 会話の表示エリア(ユーザーとAIのやり取りが吹き出しで表示される)
  - メッセージ入力欄(画面の下にある入力ボックス)

### 3.2 メッセージの送受信
- 入力欄にメッセージを書いてEnterを押すとAIに送られる
- 送ったメッセージは「ユーザー」の吹き出しとして表示される
- AIからの返事は「アシスタント」の吹き出しとして表示される

### 3.3 会話の記憶
- これまでのやり取りを全てAIに送るので、AIは話の流れを覚えている
- 例:「さっき聞いたことの続きだけど…」と言えば、AIは前の話を踏まえて答える
- Streamlitのsession_state(一時的な記憶領域)に会話履歴を保存する

### 3.4 APIキーの安全管理
- APIキーは.envファイルに書いておく
- .envファイルはGitにアップロードされない(.gitignoreで除外)
- プログラムは起動時に.envファイルからAPIキーを自動で読み込む

## 4. プログラムの構成(chat_2.py)

```
chat_2.py の中身:
├── ファイル先頭のコメント(プログラムの説明)
├── 必要な部品の読み込み(import)
├── .envファイルからAPIキーを読み込む処理
├── Claude APIの準備
├── ページの設定(タイトルなど)
├── 会話履歴の初期化
├── 過去のメッセージを画面に表示する処理
├── ユーザーの入力を受け取る処理
├── Claude APIにメッセージを送る関数
└── AIの返答を画面に表示する処理
```

### 処理の流れ

```
1. プログラム起動(streamlit run chat_2.py)
     ↓
2. .envファイルからAPIキーを読み込む
     ↓
3. ブラウザにチャット画面を表示
     ↓
4. ユーザーがメッセージを入力
     ↓
5. 入力されたメッセージを会話履歴に追加
     ↓
6. 会話履歴をまとめてClaude APIに送信(← これが「記憶」の仕組み)
     ↓
7. AIからの返答を受け取る
     ↓
8. 返答を会話履歴に追加して画面に表示
     ↓
9. 4に戻る(繰り返し)
```

## 6. 必要な準備

### requirements.txt の内容
```
anthropic
streamlit
python-dotenv
```

### 部品のインストール方法
```
pip install -r requirements.txt
```

### .envファイルの準備
```
ANTHROPIC_API_KEY=ここにAPIキーを貼り付ける
```

### プログラムの起動方法
```
streamlit run chat_2.py
```
起動すると自動的にブラウザが開き、チャット画面が表示される。

## 7. 注意すること

- APIキーは絶対に他人に見せないこと(.envファイルをGitにアップしない)
- インターネット接続が必要(Claude AIと通信するため)
- API利用には料金がかかる場合があるので、使いすぎに注意
- ブラウザを閉じると会話履歴は消える(次回起動時はリセットされる)
