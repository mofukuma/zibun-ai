# コメダ珈琲店情報チャット インターネット検索機能追加 計画書

---

## 1. 要件（何を実現するか）

このプロジェクトでは、現在のコメダ珈琲店情報チャットアプリに「**インターネット検索機能**」を追加します。

### 実現したいこと

- **店舗情報だけでなく、インターネット上の最新情報も検索できるようにする**
  - 例：「コメダ珈琲の新メニューは？」「コメダ珈琲のキャンペーン情報は？」など

- **画面上でインターネット検索のON/OFFを切り替えられる**
  - 店舗情報だけで回答してほしい時は「OFF」
  - 最新情報も含めて回答してほしい時は「ON」
  - ユーザーが自由に使い分けられる

- **既存の機能はそのまま維持する**
  - 店舗情報検索機能は変更せず、新機能として追加する

---

## 2. 使用する技術

### 追加する技術

#### **Tavily Search（タビリー サーチ）**
インターネット検索を行うためのサービスです。

- **役割**: インターネット上の情報を検索して取得する
- **特徴**:
  - 検索結果が整理されて返ってくる
  - 最新の情報を取得できる
  - 使い方がシンプル

- **使用方法**:
  ```python
  from tavily import TavilyClient

  # クライアントを作成
  client = TavilyClient(api_key="あなたのAPIキー")

  # 検索を実行
  result = client.search("検索したいキーワード")
  ```

### 既存の技術（変更なし）

以下の技術は今まで通り使用します：

- **Streamlit**: 画面を作成するツール
- **LlamaIndex**: 店舗情報を検索するツール
- **Claude Sonnet 4**: 回答を生成するAI
- **HuggingFace**: テキストを解析するツール

## 3. 追加する機能

### 機能1: インターネット検索のON/OFF切り替え

**どこに配置するか**:
- 画面左側のサイドバーに配置します
- チェックボックスで「インターネット検索を使用する」を表示

**動作**:
- チェックを入れる（ON）→ インターネット検索を使用
- チェックを外す（OFF）→ 店舗情報のみで回答

### 機能2: インターネット検索の実行

**どのように動くか**:

1. **ユーザーが質問を入力**
   - 例：「コメダ珈琲の新メニューは？」

2. **インターネット検索がONの場合**
   - Tavily Searchで質問に関連する情報をインターネットから検索
   - 検索結果を取得（最大5件）

3. **AIが回答を生成**
   - 店舗情報 + インターネット検索結果を組み合わせる
   - 総合的な回答を生成する

4. **回答を画面に表示**
   - ユーザーにわかりやすく表示

### 機能3: 検索結果の表示

**何を表示するか**:
- インターネット検索を使用した場合、どの情報源から回答したかを表示
- 例：「以下の情報源を参考にしました」として、URLのリストを表示

## 6. APIキーの管理

### .env ファイルに追加

Tavily SearchのAPIキーを環境変数として管理します。

**.env ファイルに追加する内容**:

**既存の.envファイル**:
