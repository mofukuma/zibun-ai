# Claude AIチャットプログラム 開発計画書

## 1. 要件(このプログラムで実現したいこと)

### 基本要件
- ブラウザ上でClaude AIと日本語で対話できること
- メッセージを入力して送信し、AIからの返答を受け取れること
- 過去の会話履歴が画面上に表示されること
- シンプルで使いやすい画面であること

### 技術的要件
- プログラムは1つのファイル(chat_1.py)にまとめること
- ファイルの先頭、関数の前、複雑な処理の前に日本語のコメントを入れること
- Claude APIを使ってAIとやり取りすること
- ローカル環境(自分のパソコン)で動作すること

## 2. 使用する技術

### プログラミング言語
- **Python** - AIとのやり取りやウェブサーバーの機能を実装

### 主な技術・ライブラリ
- **anthropic** - Claude AIと通信するための公式ライブラリ
- **Flask** - Pythonでウェブアプリケーションを作るための軽量フレームワーク
- **HTML/CSS/JavaScript** - ブラウザに表示する画面を作成

### API
- **Claude API** - Anthropic社が提供するAI機能を利用するためのサービス
  - APIキー(パスワードのようなもの)が必要

## 3. 機能一覧

### 3.1 画面表示機能
- ブラウザでアクセスすると、チャット画面が表示される
- 画面には以下の要素がある:
  - タイトル表示
  - 会話履歴表示エリア(過去のやり取りが表示される)
  - メッセージ入力欄(ユーザーが文章を入力する場所)
  - 送信ボタン(メッセージをAIに送るボタン)

### 3.2 メッセージ送信機能
- 入力欄にメッセージを書いて送信ボタンを押すと、AIに質問が送られる
- 送信したメッセージは会話履歴に「ユーザー」として表示される

### 3.3 AI応答機能
- Claude APIにメッセージを送信する
- AIからの返答を受け取る
- 受け取った返答を会話履歴に「AI」として表示する

### 3.4 会話履歴管理機能
- ユーザーとAIのやり取りを順番に保存する
- 画面を更新しても会話が消えないようにする(セッション管理)

## 4. プログラム構成(chat_1.py)

### 4.1 ファイル構成
```
chat_1.py (1ファイルに全て含む)
├── 必要なライブラリの読み込み
├── 設定(APIキーなど)
├── Flaskアプリケーションの初期化
├── 会話履歴を保存する仕組み
├── トップページを表示する関数
├── メッセージを送信する関数
├── Claude APIと通信する関数
└── プログラム起動部分
```

### 4.2 主要な処理の流れ

1. **プログラム起動**
   - Flaskサーバーを起動
   - ブラウザで http://localhost:5000 にアクセス

2. **画面表示**
   - HTMLでチャット画面を生成
   - CSSで見た目を整える
   - JavaScriptで送信ボタンの動作を設定

3. **メッセージ送信から応答まで**
   - ユーザーがメッセージを入力して送信
   - サーバーがメッセージを受け取る
   - Claude APIに質問を送信
   - AIからの返答を受け取る
   - 会話履歴に追加
   - ブラウザに返答を返す
   - 画面に表示

## 5. 必要な準備

### 5.1 インストールが必要なもの
```
pip install anthropic flask
```

### 5.2 APIキーの取得
- Anthropic社のウェブサイトでアカウント作成

## 6. 開発の流れ

1. **基本構造の作成**
   - Flaskアプリケーションの初期化
   - トップページの表示機能

2. **画面の作成**
   - HTMLで画面レイアウト
   - CSSで装飾

3. **AI通信機能の実装**
   - Claude APIとの接続
   - メッセージ送受信機能

4. **会話履歴機能の実装**
   - セッション管理
   - 履歴の表示

5. **動作確認とテスト**
   - 実際に動かして確認
   - エラーがあれば修正

## 7. 注意事項

- APIキーは外部に公開しないこと
- インターネット接続が必要(Claude APIと通信するため)
- API利用には料金がかかる場合があるため、使用量に注意
- エラーが発生した場合の処理を適切に行うこと
