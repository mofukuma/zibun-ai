# じぶんAIのつくりかた 〜Claude CodeではじめるRAG入門〜

書籍『じぶんAIのつくりかた 〜Claude CodeではじめるRAG入門〜』のサポートサイトです。

## 実行方法

```bash
git clone https://github.com/mofukuma/zibun-ai.git
cd zibun-ai

pip install -r requirements.txt

# APIキーを設定（.envファイルを作成）
echo "ANTHROPIC_API_KEY=your_api_key_here" > .env
```

**APIキーの取得:** [Anthropic Console](https://console.anthropic.com/) でアカウントを作成してAPIキーを発行してください。
